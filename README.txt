FxzzStore Pro

تشغيل المشروع:
1) افتح المجلد في VS Code
2) افتح Terminal من نوع Command Prompt وليس PowerShell
3) نفّذ:
   npm install
   npm start
4) افتح:
   http://localhost:3000
